Description of tools, frameworks, and API's used:

In this project, we implemented many of the tools and approaches learned from class. We have a more detailed list of how these tools, frameworks, and API's are used in our project. Thanks!

  • Vue: Used to build the UI and help consturct the single-page application seen on the homepage for the map and table functionality.
  • Project API: Used to get the data from the stpaul_crime database in formats we can use.
  • Leaflet: Used to create the map and the functionality of the map (zoom, pins, and panning).
  • Foundation: Used to create the layout of the website and for some styling.
  • jQuery: Used to to help with event handling and manipulating the DOM.
  • Node.js: Used for tracking dependencies and for initiating our server.
  • Nomination: Used to get the latitude and longitude of addresses and positions of the map.
